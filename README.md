# GRC Job Simulations

This repository contains cybersecurity governance, risk, compliance (GRC), ITGC audit, and risk management job simulation projects completed through professional virtual experience programs.

## Objectives

* Practice ITGC auditing concepts
* Perform risk assessments
* Review control design and effectiveness
* Analyze SDLC and change management processes
* Develop audit-style documentation
* Build a portfolio demonstrating GRC and cybersecurity governance skills

## Simulations Included

### PwC Cybersecurity Consulting

* Task 1 – Procure-to-Pay (P2P) Risk Assessment
* Task 2 – SDLC Walkthrough
* Additional tasks to be added as completed

## Skills Demonstrated

* IT General Controls (ITGC)
* Risk Assessment
* Control Testing
* Audit Documentation
* SDLC Reviews
* Compliance Analysis
* Governance, Risk & Compliance (GRC)
